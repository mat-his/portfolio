Android App

Le 05 Janvier 2021

I am currently working on an Android application allowing its users to develop and program robots from programmable cards such as Raspberry Pi. The goal of this application is to be able to find all the functions and advantages of a PC on your smartphone , in an ergonomic and synthetic way, in order to fully use the capacities of machines that we carry every day in our pocket.

The application establishes a working environment that allows you to write down your ideas, to diagram the connections and assemblies (in 2D and 3D), to write code and send it to the programmable card, and even to pilot the robot that the user will have built through different types of interactions. the software uses wireless protocols and connections such as SSH, SFTP, Bluetooth and port programming to communicate with the robot, defined as a server.

As a bonus, this data is stored in a real-time database that allows multiple people to collaborate, share some work with a community of people working around the same subject, or find his data on other devices such as on his PC.


The application is initially written in Kotlin, but switches to Flutter to be able to extend to Linux OS, iOS and Web.


The design, graphics as well as development details will be visible very soon once the application published!
